== Description ==
 
Frontier Restrict media will restrict users to only access their own media files 

= Main Features =
* Users without the capability edit_others_posts will only have access to their own media files from the media gallery
 
Related plugins: [Frontier Post](http://wordpress.org/plugins/frontier-post/) , [Frontier Buttons](http://wordpress.org/plugins/frontier-buttons/) 
 
== Installation ==

1. ftp the plugin to the `/wp-content/plugins/`  directory or install it from plugin install on your site
2. Activate the plugin through the 'Plugins' menu in WordPress
